BomberBoy

BomberBoy is a (very) little BomberMan-clone. It's just a fun little side project to practice javascript and responsive design with touch controls.
There are 3 simple levels that repeat. Have fun!... or not. I'm just a paragraph, not a cop.

If you are checking touch screen options in the developer tools please reload after the change. The page only checks touch options on load.
